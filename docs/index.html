About the game

Vampire Killer is an action platformer game from the Castlevania series created by Konami. In this single-player game, you control Simon Belmont through the different levels of Dracula's castle in order to defeat Lord Dracula and bring peace to Transylvania once again. This horizontal 2D game is divided into 18 stages, in which the player will have to progress fighting various enemies with one weapon, the Vampire killer whip. Once we reach the end of the final stage, there is a boss fight in which we must fight Lord Dracula, the final boss. To reach the end, the player will have to find gather master keys in order to progress through the levels. Said master keys are well hidden in different castle rooms, but luckily there are more keys hidden around the castle, which open treasure chests that contain boost objects that can help the player. These boost objects can also be obtained from several merchants the player will find along the way, or hidden behind breakable walls.
 
We have tried to recreate a bit of the MSX2 (1987) version including the entrance of the castle and the second level, the laboratory. In this tribute the objective is to beat the second level of the game and face its main boss Medussa to finish the game and reach the end of the story.
 
 

Gameplay

⇒ Vampire Killer MSX2

Controls

  • Arrow keys for player movement. (↑→↓←)

 

Space bar to transition through game screens (Initial screen, Title screen, Game stages, Victory Credits and Game Over scenes)  or Attack.

Debug Keys:

  • F1: Show colliders

 

  • M: Takes player’s lifes automatically.

 

  • W: Automatic victory.

 

  • D: Automatic defeat.

Level Debug:

  • Press key 0: Load level 1

 

  • Press key 1: Load level 2

 

  • Press key 2: Load level 3

 

  • Press key 3: Load level 4

Enemy Debug:

  • Press key 5: Spawn Soldier

 

  • Press key 6: Spawn SkullTurret

 

  • Press key 7: Spawn Medusa

Object Debug:

  • Press key 8: Spawn Heart

 

  • Press key 9: Spawn White Money Bag
  • Press key Left shift: Spawn Boss Orb
